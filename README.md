# 🤖 Telegram-бот «База регистраций» (Node.js)

## Стек
- **Node.js** 18+
- **Telegraf** v4 — фреймворк для Telegram-ботов
- **better-sqlite3** — быстрая синхронная SQLite БД
- **dotenv** — переменные окружения

---

## Установка и запуск

### 1. Клонируйте / распакуйте проект

```bash
cd tgbot
```

### 2. Установите зависимости

```bash
npm install
```

### 3. Настройте `.env`

```bash
cp .env.example .env
```

Откройте `.env` и заполните:

```env
BOT_TOKEN=токен_от_@BotFather
ADMIN_IDS=ваш_telegram_id
MANAGER_USERNAME=@username_менеджера
```

> Узнать свой Telegram ID можно через @userinfobot  
> Несколько админов: `ADMIN_IDS=111111,222222`

### 4. Запуск

```bash
# Обычный запуск
npm start

# Режим разработки (авто-перезапуск при изменениях, Node 18+)
npm run dev
```

## Структура проекта

```
tgbot/
├── src/
│   ├── bot.js           # Точка входа, регистрация хендлеров
│   ├── config.js        # Конфигурация из .env
│   ├── database.js      # Вся работа с SQLite (better-sqlite3)
│   ├── keyboards.js     # Все клавиатуры (reply + inline)
│   ├── session.js       # Хранилище FSM-состояний в памяти
│   └── handlers/
│       ├── user.js      # Хендлеры для пользователей
│       └── admin.js     # Хендлеры для администраторов
├── package.json
├── .env.example
└── bot.db               # SQLite БД (создаётся автоматически)
```

## Функционал

### Для пользователей
| Функция | Описание |
|---|---|
| 🔐 Вход | Только по реф-ссылке или инвайт-коду |
| 📸 Верификация | Отправка фото → одобрение администратором |
| 📋 Сайты | Список с оплатой, статусами, переход по ссылке |
| ✅ Готово | Подача заявки на проверку регистрации |
| 💰 Баланс | Общий / с регистраций / с рефералов |
| 👥 Рефералы | Уникальная ссылка, статистика |
| 💳 Кошелек | Привязка и изменение USDT TRC20 |
| 💸 Вывод | Заявка администратору на выплату |

### Для администратора (`/admin`)
| Раздел | Возможности |
|---|---|
| 🌐 Сайты | Добавить, редактировать, включить/отключить |
| 📝 Регистрации | Подтвердить / отклонить (баланс начисляется автоматически) |
| 💸 Выводы | Подтвердить / отклонить (при отклонении баланс возвращается) |
| 🔐 Верификации | Просмотр фото, одобрение / отказ |
| 🎟 Инвайт-коды | Создание одноразовых кодов для входа |
| 👥 Пользователи | Список всех пользователей с балансами |

## Реферальная система

- Каждый пользователь получает уникальную ссылку:  
  `https://t.me/ВАШ_БОТ?start=ref_123456789`
- При подтверждении регистрации реферала — рефереру начисляется % автоматически
- Процент задаётся отдельно для каждого сайта в админ-панели

## Деплой на сервер (VPS)

### Через PM2

```bash
npm install -g pm2
pm2 start src/bot.js --name tgbot
pm2 save
pm2 startup
```

### Через systemd

```ini
# /etc/systemd/system/tgbot.service
[Unit]
Description=Telegram Registration Bot
After=network.target

[Service]
WorkingDirectory=/путь/до/tgbot
ExecStart=/usr/bin/node src/bot.js
Restart=always
EnvironmentFile=/путь/до/tgbot/.env

[Install]
WantedBy=multi-user.target
```

```bash
systemctl enable tgbot
systemctl start tgbot
```
